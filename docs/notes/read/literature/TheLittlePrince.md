# 前言
这是我第一次完整的读完这本书  
在书柜上躺过我人生少说十载春秋，从未完整的读完过一遍  
如今终于是完整的读完一遍。作为一本世界上享有盛名的童话，在儿时阅读时却只会感到乏味和枯燥，现在长大了，我貌似已经变成书中所说的“无聊的大人”，面对小时候认为荒诞且枯燥的故事，我总是站在作者想要表达的对立面去思考问题。我像故事开端的飞行员，不能透过箱子看到里面的绵羊。
# 每个人都住在自己的小行星里
::: note
“从前有个小王子，他住在小行星里。这颗行星比他自己大不了多少。他需要一个朋友”
:::
该去交些朋友了。
小王子开启了自己的旅途，先去拜访别的小行星吧  
自认为统治一切的国王，爱慕虚荣的人，为了忘记而一直喝酒的酒鬼，认为拥有所有星星的商人，恪守规矩的点灯人，地理学家，失事的飞行员  

小王子可以在自己的世界看四十三次日落，国王也可以在自己的行星统治一切，爱慕虚荣的人
