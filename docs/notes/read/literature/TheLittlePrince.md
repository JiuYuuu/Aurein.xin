# 前言
这是我第一次完整的读完这本书  
在书柜上躺过我人生少说十载春秋，从未完整的读完过一遍  
如今终于是完整的读完一遍。作为一本世界上享有盛名的童话，在儿时阅读时却只会感到乏味和枯燥，现在长大了，我貌似已经变成书中所说的“无聊的大人”，面对小时候认为荒诞且枯燥的故事，我总是站在作者想要表达的对立面去思考问题。我像故事开端的飞行员，不能透过箱子看到里面的绵羊。
